# /learn - Start Guided Learning

Start a structured learning path with agent guidance and skill development.

## Usage

```
/learn                      # Start learning wizard
/learn react                # Start React learning path
/learn backend              # Start Backend learning path
/learn {role-name}          # Start learning for any role
/learn assessment           # Take skills assessment
```

## Description

This command initiates:
- **Learning path selection** wizard
- **Personalized roadmap** based on your level
- **Daily learning plans** and schedules
- **Progress tracking** across skills
- **Agent-guided tutoring** for each topic
- **Project assignments** at each milestone

## Tips

- **Consistency beats intensity** - Daily practice > occasional marathons
- **Build projects** alongside theory
- **Take breaks** - Your brain needs rest
- **Join cohorts** - Learn with others
- **Review regularly** - Reinforce concepts
- **Don't skip fundamentals** - They matter!
