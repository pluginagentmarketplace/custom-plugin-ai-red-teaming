# /explore - Discover Developer Roles

Explore and discover all 65+ developer roles available in the Developer Roadmap ecosystem.

## Usage

```
/explore                    # List all role categories
/explore frontend           # List all frontend roles
/explore backend            # List all backend roles
/explore careers            # Show career paths
/explore comparison         # Compare roles
```

## Description

This command helps you:
- **Discover roles** across all categories
- **Compare roles** side-by-side
- **Understand career paths** and progressions
- **Get role details** with learning requirements
- **Explore specializations** within a domain
