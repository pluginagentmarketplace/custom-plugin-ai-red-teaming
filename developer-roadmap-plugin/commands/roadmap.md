# /roadmap - View Role Roadmaps

View detailed roadmaps for specific roles with step-by-step learning paths.

## Usage

```
/roadmap react              # React developer roadmap
/roadmap backend            # Backend developer roadmap
/roadmap fullstack          # Full-stack developer roadmap
/roadmap {role-name}        # Any of 65+ roles
```

## Description

This command provides:
- **Step-by-step learning path** for any role
- **Prerequisites and foundations** needed
- **Core skills and technologies** to master
- **Advanced specializations** and options
- **Project milestones** and timelines
- **Resource recommendations**
