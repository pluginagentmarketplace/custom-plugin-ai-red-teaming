# /progress - Track Learning Progress

Track your learning journey, view achievements, and get personalized recommendations.

## Usage

```
/progress                   # View overall progress
/progress {role-name}       # View progress for specific role
/progress stats             # Detailed statistics
/progress recommendations   # Get next step recommendations
/progress reset             # Reset progress (start over)
```

## Description

This command helps you:
- **Track completion** across modules
- **View achievements** and milestones
- **Understand learning velocity** and pacing
- **Get personalized recommendations** for next steps
- **Compare progress** with goals
- **Celebrate achievements** and stay motivated

## Tips

- **Track consistently** - Daily check-ins help you stay accountable
- **Celebrate wins** - Acknowledge milestones, even small ones
- **Adjust pace** - If behind, consider adjusting timeline
- **Build projects** - The most important indicator of progress
- **Stay consistent** - Streaks help build momentum
- **Review regularly** - Don't skip the fundamentals
